# Data Analysis — 数据分析

执行数据分析，强制使用代码，输出代码和结果。

## 流程

1. 确认数据来源（来自哪篇文献、哪个数据集）
2. 确认计算方法（人类已确认的公式/模型）
3. 如果公式不确定 → 输出 `[待人类确认]` 标注，停止
4. 编写 Python/R 代码执行计算
5. 输出代码 + 结果
6. 调用 `data-verifier` 代理验证

## Arguments

`$ARGUMENTS` — 分析描述（如 "计算描述性统计"、"回归分析"）

## 输出

- `output/data/analysis.py` — 计算代码
- `output/data/results.txt` — 计算结果
- 数据来源标注
- 审查清单

## 铁律

- **所有计算必须用代码** — 禁止心算、估算、编造数字
- **输出代码 + 结果** — 人类需要看到完整的计算过程
- **不会的公式 → 标注空位** — 绝不编造公式
- **数据来源必须标注** — 每个数字都要有出处

## 注意

- 公式不确定时停止，等待人类确认
- 数据来源必须标注
- 代码必须可运行、可审查
