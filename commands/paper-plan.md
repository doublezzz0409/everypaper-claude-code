# Paper Plan — 论文规划

规划论文结构，输出大纲和待确认事项。

## 流程

1. 读取已有的研究主题和文献
2. 调用 `paper-planner` 代理
3. 输出论文结构规划
4. 标注所有待人类确认的事项
5. 等待人类审查

## Arguments

`$ARGUMENTS` — 可选：指定已有论文项目目录

## 输出

- 论文结构大纲
- 待确认事项清单（方法、变量、数据来源）
- 建议（基于学术规范）

## 注意

- 研究方向由人类决定
- 分析方法不确定时标注 `[待人类确认]`
- 人类确认后才能继续下一步
