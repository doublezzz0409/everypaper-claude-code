# Data Verification — 数据验证

## When to Use

- 执行数值计算时
- 引用数据时
- 写结论时
- `/data-analysis` 命令

## How It Works

### 铁律

1. **所有计算必须用代码** — 禁止心算、估算、编造数字
2. **输出代码 + 结果** — 人类需要看到完整的计算过程
3. **不会的公式 → 标注空位** — 绝不编造公式
4. **数据来源必须标注** — 每个数字都要有出处

### 计算流程

1. 确认数据来源（来自哪篇文献、哪个数据集）
2. 确认计算方法（人类已确认的公式/模型）
3. 如果公式不确定 → 输出 `[待人类确认]` 标注，停止
4. 编写 Python/R 代码执行计算
5. 输出代码 + 结果
6. 调用 `data-verifier` 代理验证

### 输出格式

```
## 计算结果

### 数据来源
- 文件: papers/zhang2023.xlsx
- 表: Table3
- DOI: 10.xxxx/xxxxx

### 计算代码
[完整 Python 代码]

### 计算结果
[代码输出]

### 审查清单
- [ ] 数据来源是否正确？
- [ ] 公式是否经过人类确认？
- [ ] 结果是否合理？
```

## Examples

```
用户: 计算描述性统计

我: 确认数据来源：
    - 文件: papers/zhang2023.xlsx
    - 表: Table3
    
    请确认：
    1. 数据来源是否正确？
    2. 需要计算哪些统计量？
```
