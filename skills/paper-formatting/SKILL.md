# Paper Formatting — 论文格式化

## When to Use

- 写论文时
- 格式化引用时
- 编号图表时
- `/paper-export` 命令

## How It Works

### 引用格式

#### 文内引用
- 单篇引用：(Zhang, 2023)
- 多篇引用：(Zhang, 2023; Li, 2022)
- 作者名引用：Zhang (2023) 发现...

#### 参考文献格式
- APA 格式
- Chicago 格式
- 期刊特定格式

### 图表编号

- 图：Figure 1, Figure 2, ...
- 表：Table 1, Table 2, ...
- 每个图表必须有标题
- 每个图表必须在正文中引用

### 章节结构

```
1. 引言
2. 文献综述
3. 研究方法
4. 实证结果
5. 讨论
6. 结论
参考文献
附录
```

### 格式检查清单

- [ ] 引用格式是否一致？
- [ ] 图表是否编号并有标题？
- [ ] 参考文献格式是否统一？
- [ ] 是否符合目标期刊要求？

## 注意

- 格式由人类确认
- 不同期刊可能有不同要求
- 导出前需要人类确认格式
