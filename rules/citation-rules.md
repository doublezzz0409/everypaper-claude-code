# 引用规范

## 铁律

**引用必须有来源，可在 papers/ 找到原文。**

## 引用规则

### 文内引用

- 单篇引用：(Zhang, 2023)
- 多篇引用：(Zhang, 2023; Li, 2022)
- 作者名引用：Zhang (2023) 发现...

### 引用来源

- 引用必须在 `papers/` 有原文
- 引用必须有 DOI 或 URL
- 不确定的引用标注 `[来源待确认]`

### 引用格式

- APA 格式
- Chicago 格式
- 期刊特定格式

### 引用检查清单

- [ ] 每个引用是否可在 `papers/` 找到原文？
- [ ] 引用格式是否一致？
- [ ] 是否存在过度引用（每句超过3篇）？
- [ ] 是否存在遗漏引用（关键观点无支撑）？
- [ ] 引用的文献是否权威？
