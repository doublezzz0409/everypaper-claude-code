# 禁止编造规则

## 铁律

**绝不编造任何内容。**

## 具体规则

### 文献引用

- 引用必须在 `papers/` 有原文
- 不确定的引用标注 `[来源待确认]`
- 编造文献 = 学术造假

### 数据

- 每个数字必须有来源
- 计算必须用代码执行
- 不确定的数据标注 `[数据待确认]`

### 方法

- 分析方法必须有权威来源
- 不确定的方法标注 `[待人类确认]`
- 编造方法 = 结论错误

### 结论

- 结论必须由数据支撑
- 不确定的结论标注 `[待确认]`
- 编造结论 = 学术不端

## 检查清单

- [ ] 每个引用是否可在 `papers/` 找到原文？
- [ ] 每个数字是否有来源？
- [ ] 每个方法是否有权威来源？
- [ ] 每个结论是否由数据支撑？
- [ ] 是否存在编造内容？
