# 数据处理规范

## 铁律

**所有计算必须用代码执行，禁止心算和编造。**

## 数据规则

### 数据来源

- 每个数字必须有来源
- 数据必须来自 `papers/` 中的文献
- 不确定的数据标注 `[数据待确认]`

### 数据计算

- 所有计算必须用 Python/R 代码
- 输出代码 + 结果
- 禁止心算、估算、编造数字

### 数据验证

- 调用 `data-verifier` 代理验证
- 输出审查清单
- 人类审查确认

### 数据检查清单

- [ ] 每个数字是否有来源？
- [ ] 计算是否有代码记录？
- [ ] 结果是否合理？
- [ ] 是否存在数据编造嫌疑？
