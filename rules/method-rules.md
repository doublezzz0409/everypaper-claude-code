# 方法论规范

## 铁律

**方法不能乱编，不会的空着标注。**

## 方法规则

### 方法来源

- 分析方法必须有权威来源
- 来源必须是教科书或权威论文
- 不确定的方法标注 `[待人类确认]`

### 方法确认

- 我提供方法建议（基于学术规范）
- 人类确认是否使用
- 不确定的方法标注空位
- 人类提供权威来源后补全

### 公式处理


### 方法检查清单

- [ ] 分析方法是否有权威来源？
- [ ] 公式是否来自教科书或权威论文？
- [ ] 不确定的方法是否标注了 `[待人类确认]`？
- [ ] 人类是否确认了分析方法？
